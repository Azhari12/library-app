## Features

- Show book availabe book list
- Can rent Book
- Can show rented book user



## Tech Stack

**Client:** React, Redux, TailwindCSS

**Server:** Node


## Run Locally

Clone the project

```bash
  git clone https://github.com/Azhari12/library-app.git
```

Go to the project directory

```bash
  cd library-app
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
